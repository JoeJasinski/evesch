Welcome to The Event Scheduler!

This site is in extreme Alpha and any data entered into it will not be preserved until we officially launch! Feel free to test it with login "demo" password "demo".

EveSch.com is a free social network that allows organizations to coordinate events and manage these events through one simple interface.

EveSch.com provides the following features:
  • Simple tool to create and customize organizations and events
  • Calendar to plan events and export them to iCal format
  • Ability to delegate event planning to other members on the organization
  • Customizable sign-up sheets to track attendance to events
  • Reports to help track past events and the ability to track hours
  • Much more!
